# Virgilio Sánchez — Photography Portfolio

A personal portfolio website for photographer and videographer Virgilio Sánchez, based in Saltillo, México. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

## Live Demo

virgiliovisuals.netlify.app

## Features

- **Photo gallery** with masonry layout and category filters
- **Series system** — each gallery card opens a full mosaic of photos from that series
- **Video support** — video series display 16:9 thumbnails generated by Cloudinary, with full playback on click
- **Lightbox** — click any photo or video to view it full screen; press `Escape` or `✕` to go back to the mosaic
- **Keyboard navigation** — use arrow keys to move between photos inside a series
- **Smooth scroll animations** on page load using IntersectionObserver
- **Contact form** powered by Formspree — submissions go directly to email
- **Fully responsive** — works on mobile, tablet, and desktop

## Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Page structure |
| CSS3 | Styling, CSS variables, masonry layout, animations |
| JavaScript (ES6+) | Gallery logic, lightbox, filters, form handling |
| [Cloudinary](https://cloudinary.com) | Image and video hosting, automatic optimization (`q_auto`, `f_auto`), video thumbnails |
| [Formspree](https://formspree.io) | Contact form email delivery |
| [Google Fonts](https://fonts.google.com) | Cormorant Garamond + Jost typefaces |

## Adding a New Photo Series

Open `main.js` and add a new entry to the `series` object:

```javascript
"category": {
  titulo: "Category Name",
  series: {
    "series-name": {
      portada: `${CLOUD}/photo_cover.jpg`,  // cover shown in gallery
      titulo: "Series Title",
      fotos: [
        `${CLOUD}/photo_1.jpg`,
        `${CLOUD}/photo_2.jpg`,
      ]
    }
  }
}
```

Then add the filter button in `index.html`:

```html
<button class="filter-btn" data-cat="category">Category</button>
```

## Adding a New Video Series

```javascript
"videos": {
  titulo: "Videos",
  series: {
    "series-name": {
      portada: `${CLOUD}/cover_image.jpg`,
      titulo: "Series Title",
      videos: [
        { src: `${CLOUD_VIDEO}/video_1.mp4`, segundo: 3 },
        { src: `${CLOUD_VIDEO}/video_2.mp4`, segundo: 5 },
      ]
    }
  }
}
```

The `segundo` value sets which second of the video is used as the thumbnail.

## Running Locally

No build step required. Just open `index.html` in your browser or use a local server:

```bash
npx serve .
```
